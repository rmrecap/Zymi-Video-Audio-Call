# ZYMI Closed Beta — Tester Rules

**Effective:** 2026-06-02  
**Scope:** All closed beta testers (Groups A–E)

---

## 1. Confidentiality

| Rule | Description |
|------|-------------|
| **1.1** | Do not share the APK file, IPA file, or beta URL with anyone outside the beta program. |
| **1.2** | Do not share screenshots, screen recordings, or videos of the beta app on social media, forums, or any public channel. |
| **1.3** | Do not discuss unreleased features or bugs publicly. |
| **1.4** | Beta invite messages are confidential — do not forward them. |

## 2. Acceptable Use

| Rule | Description |
|------|-------------|
| **2.1** | Do not upload, share, or transmit illegal content (including copyrighted material without permission). |
| **2.2** | Do not upload violent, hateful, sexually explicit, or harassing content. |
| **2.3** | Do not abuse the call or messaging system (no spam, automated messages, or harassment). |
| **2.4** | Do not attempt to hack, exploit, or reverse-engineer the app. |
| **2.5** | Do not create multiple accounts. |

## 3. Bug Reporting

| Rule | Description |
|------|-------------|
| **3.1** | Report bugs as soon as you find them. |
| **3.2** | Include steps to reproduce — the more detail, the faster we can fix it. |
| **3.3** | Include screenshots or screen recordings if possible. |
| **3.4** | Do not file duplicate bugs — check existing bug reports first. |
| **3.5** | Do not publicly disclose security vulnerabilities. Report them privately to [admin-email-placeholder@zymi.app]. |

## 4. Respect

| Rule | Description |
|------|-------------|
| **4.1** | Treat other testers with respect. |
| **4.2** | No harassment, bullying, or discrimination. |
| **4.3** | No hate speech or offensive language. |
| **4.4** | No spamming testers or admins. |

## 5. Consequences

| Violation | Consequence |
|-----------|-------------|
| First offense (minor) | Warning from admin |
| Repeated minor offenses | Temporary suspension (3 days) |
| Major violation (abuse, illegal content, hacking) | **Immediate and permanent ban** |
| Public disclosure of beta | **Immediate and permanent ban** |
| Harassment of other testers | **Immediate and permanent ban** |

## 6. Acknowledgment

By participating in the ZYMI Closed Beta, I agree to:

1. Follow all rules listed above
2. Report bugs and submit feedback in good faith
3. Keep beta materials confidential
4. Accept that the beta software may contain bugs and data loss is possible
5. Accept that my account and data may be reset during the beta period
